# How to use it

## Mac version:
[Download](https://github.com/OOOHA/Wake/releases/download/V1/wake.zip)

<span style="color: orange; font-size: 1em;">⚠️</span> **Note: The file is downloaded as a ```zip``` file, please unzip it first.**

### Run the App
![image](https://github.com/OOOHA/Wake/blob/main/img/mac1.png)

Click on the start Preventing sleep

![image](https://github.com/OOOHA/Wake/blob/main/img/maca.png)

Click OK when seeing this popup

![image](https://github.com/OOOHA/Wake/blob/main/img/mac2.png)

You can now see that sleep prevention is active.<br>
When you're done, click stop Preventing sleep.<br>
Or you can<br>
* quit the app.
* use ```command + Q``` to quit the app.

![image](https://github.com/OOOHA/Wake/blob/main/img/macs.png)

<span style="color: orange; font-size: 1em;">⚠️</span> **Note: You may need to permit to open this app for the first time.**
![image](https://github.com/OOOHA/Wake/blob/main/img/macd1.png)

Click OK and go to System Settings -> Privacy and Security

![image](https://github.com/OOOHA/Wake/blob/main/img/macd2.png)

Find this section in the Privacy and Security -> click open Anyway

![image](https://github.com/OOOHA/Wake/blob/main/img/macd3.png)

Click Open and enter your password or use TouchID.

# 使用方法

## Mac 版:
[下載](https://github.com/OOOHA/Wake/releases/download/V1/wake.zip)

<span style="color: orange; font-size: 1em;">⚠️</span> **注意：如下載後為壓縮檔，請先解壓縮**

### 執行程式
![image](https://github.com/OOOHA/Wake/blob/main/img/mac1.png)

點擊 start Preventing sleep

![image](https://github.com/OOOHA/Wake/blob/main/img/maca.png)

在彈出視窗中點擊OK

![image](https://github.com/OOOHA/Wake/blob/main/img/mac2.png)

現在防睡眠已啟用.<br>
如果要結束防睡眠可以點擊stop Preventing sleep.<br>
或是可以透過以下方法結束程式<br>
* 用Mac本身的結束程式.
* 使用 ```cammand + Q``` 結束程式.

![image](https://github.com/OOOHA/Wake/blob/main/img/macs.png)

<span style="color: orange; font-size: 1em;">⚠️</span> **注意: 第一次開啟時須提供權限.**

![image](https://github.com/OOOHA/Wake/blob/main/img/macd1.png)

點擊 OK 並前往 系統設定 -> 隱私與安全性

![image](https://github.com/OOOHA/Wake/blob/main/img/macd2.png)

在隱私與安全性頁面中找到該畫面，點擊強制開啟

![image](https://github.com/OOOHA/Wake/blob/main/img/macd3.png)

點擊開啟並輸入密碼或是使用指紋解鎖(TouchID)

## Windows 版:
[下載](https://github.com/OOOHA/Wake/releases/download/V1/wake.exe)

### 執行程式
![image](https://github.com/OOOHA/Wake/blob/main/img/win-app.png)

開啟程式後會看到這個視窗，關閉視窗以結束程式。
